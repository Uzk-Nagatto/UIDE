
# Pong Game

Este es un juego básico de Pong hecho en Python usando la biblioteca **Pygame**. El juego tiene dos paletas que los jugadores pueden mover para hacer rebotar la pelota. El objetivo es evitar que la pelota salga de la pantalla y conseguir un "punto" al hacer que la pelota pase al otro lado.

## Requisitos

- Python 3.6 o superior.
- Pygame (para crear la ventana y dibujar los objetos en la pantalla).

## Instalación

### 1. Instalar Python

Si no tienes Python instalado en tu computadora, puedes descargarlo desde [python.org](https://www.python.org/downloads/).

### 2. Instalar Pygame

Abre tu terminal o línea de comandos y ejecuta el siguiente comando para instalar Pygame:

```bash
pip install pygame
```

### 3. Descargar el código

Puedes copiar el código del juego de Pong o descargar el archivo `.py` de [aquí](enlace-al-archivo-pong.py).

### 4. Ejecutar el juego

Una vez que tengas Pygame instalado y el archivo de código listo, puedes ejecutar el juego con:

```bash
python pong.py
```

## Controles

- **Jugador 1**: Usa las teclas **W** (subir) y **S** (bajar) para mover la paleta.
- **Jugador 2**: Usa las teclas de las flechas **Arriba** y **Abajo** para mover la paleta.

## Funcionalidades

- **Rebotes**: La pelota rebota al tocar los bordes superior e inferior de la ventana.
- **Colisiones con las paletas**: Si la pelota toca alguna de las paletas, rebota.
- **Reinicio de la pelota**: Si la pelota sale por el lado izquierdo o derecho de la pantalla, la pelota se reinicia en el centro.

## Notas

- Este juego está diseñado para ser jugado por dos personas en un mismo teclado.
- El código es muy básico y está hecho para ayudar a los principiantes a entender cómo funciona un juego de Pong usando Pygame.
